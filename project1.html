Mata Pelajaran TIK – Kelas XI

📋 Deskripsi Project

Website Biodata Siswa adalah halaman web yang menampilkan profil lengkap seorang siswa, meliputi data diri, nilai ulangan semester, dan portofolio project. Website ini dibangun menggunakan HTML untuk struktur, CSS untuk tampilan visual yang menarik dan responsif, serta JavaScript untuk interaktivitas dasar. Project ini merupakan tugas pertama dalam mata pelajaran TIK yang bertujuan melatih kemampuan dasar pembuatan website.
