Kalkulator JavaScript adalah aplikasi web interaktif yang mampu melakukan operasi aritmatika dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian. Dibangun sepenuhnya dengan HTML, CSS, dan JavaScript tanpa menggunakan library eksternal. Project ini melatih kemampuan manipulasi DOM dan logika pemrograman JavaScript.
